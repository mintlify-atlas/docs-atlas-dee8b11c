## Overview

Codante.io uses **TailwindCSS v4** with a custom design system that includes:

- Custom color palette (brand, background, gray)
- Custom fonts (Inter, Lexend, Gloria Hallelujah)
- Custom animations and keyframes
- Dark mode support
- Responsive utilities
- Custom utility classes

## Configuration

Tailwind v4 uses CSS-based configuration in `app/tailwind.css`:

```css app/tailwind.css
@import "tailwindcss";

@import url("/fonts/fonts.css") layer(base);
@import "./typography.css" layer(components);

@custom-variant dark (&:is(.dark *));

@theme {
  /* Custom theme configuration */
}
```

## Dark Mode

Dark mode is implemented using a custom variant:

```css app/tailwind.css
@custom-variant dark (&:is(.dark *));
```

### Dark Mode Classes

```tsx
<div className="bg-white dark:bg-background-900 text-gray-900 dark:text-white">
  Automatically adapts to dark mode
</div>

<div className="border-gray-200 dark:border-background-700">
  Dark mode borders
</div>
```

### Color Mode Context

Access the color mode via React context:

```tsx
import { useColorMode } from "~/lib/contexts/color-mode-context";

function MyComponent() {
  const { colorMode } = useColorMode();
  
  return (
    <img 
      src={colorMode === "light" ? "/logo-light.svg" : "/logo-dark.svg"} 
      alt="Logo" 
    />
  );
}
```

## Custom Utilities

### Container Utility

Custom responsive container with padding:

```css app/tailwind.css
@utility container {
  padding-inline: 0.8rem;
  @media (width >= theme(--breakpoint-sm)) {
    padding-inline: 2rem;
  }
  @media (width >= theme(--breakpoint-lg)) {
    padding-inline: 1.5rem;
  }
  @media (width >= theme(--breakpoint-xl)) {
    padding-inline: 3rem;
  }
  @media (width >= theme(--breakpoint-2xl)) {
    padding-inline: 10rem;
  }
}
```

Usage:

```tsx
<div className="container mx-auto">
  Responsive container with custom padding
</div>
```

## Best Practices

<Steps>
  <Step title="Use the cn() utility">
    Always merge classes with the `cn()` utility to avoid conflicts:
    
    ```tsx
    import { cn } from "~/lib/utils/cn";
    
    <div className={cn(
      "base-class",
      condition && "conditional-class",
      props.className
    )} />
    ```
  </Step>

  <Step title="Support dark mode everywhere">
    Always provide dark mode variants:
    
    ```tsx
    <div className="bg-white dark:bg-background-900 text-gray-900 dark:text-white">
    ```
  </Step>

  <Step title="Use custom colors consistently">
    Stick to the custom color palette (brand, background) instead of default Tailwind colors.
  </Step>

  <Step title="Leverage custom animations">
    Use the predefined animations for consistency:
    
    ```tsx
    <div className="animate-float">
    <div className="animate-bg">
    ```
  </Step>

  <Step title="Apply responsive design">
    Design mobile-first, then add responsive variants:
    
    ```tsx
    <div className="text-sm md:text-base lg:text-lg">
    ```
  </Step>
</Steps>
