## Overview

Tracks (Trilhas) are curated learning paths that combine workshops, challenges, and additional resources into a structured curriculum. They provide a guided journey through a technology stack or development topic.

## Track Structure

```typescript
type Track = {
  id: string;
  name: string;
  slug: string;
  short_description: string;
  description: string;
  image_url?: string;
  status: "draft" | "published" | "soon" | "archived";
  difficulty: 1 | 2 | 3;
  duration_in_minutes: number;
  trackables: (WorkshopTrackable | ChallengeTrackable)[];
  created_at: string;
  updated_at: string;
}
```

### Track Metadata

- **Difficulty**: Rated from 1 (beginner) to 3 (advanced)
- **Duration**: Estimated time to complete the entire track
- **Status**: Publication state (draft, published, soon, archived)
- **Trackables**: Collection of workshops, challenges, and resources

## Progress Tracking

Track your completion status for each item:

```typescript
type TrackablePivot = {
  id: string;
  trackable_type: string;
  trackable_id: number;
  track_id: number;
  position: number;
  name: string;
  description: string;
  section_id: number;
}
```

### Marking Items Complete

```typescript
await toggleTrackableCompleted(trackableId, request);
// Returns: { success: "Item marcado como concluído!" }
// Or: { success: "Item desmarcado" }
```

<Info>
Click the checkbox next to any trackable item to mark it as complete. This helps you track your progress through the learning path.
</Info>

## Type Guards

Differentiate between trackable types:

```typescript
function isWorkshopTrackable(
  trackable: WorkshopTrackable | ChallengeTrackable
): trackable is WorkshopTrackable {
  return (trackable as WorkshopTrackable).lessons !== undefined;
}
```

<Info>
Use type guards to render appropriate components based on trackable type.
</Info>

## API Reference

### Get All Tracks

```typescript
await getTracks();
// Returns: Array<Track>
```

### Get Track by Slug

```typescript
await getTrack(slug, request);
// Returns: Track with full trackables
```

### Toggle Item Completion

```typescript
await toggleTrackableCompleted(trackableId, request);
// Returns: { success: string } | { error: string }
```

## Error Handling

```typescript
if (!ok) {
  return {
    error: message,
    timestamp: new Date().toISOString(),
    trackableId,
  };
}
```

</Note>
