## Overview

Workshops are comprehensive video-based courses taught by industry professionals. Each workshop consists of organized lessons with progress tracking, downloadable resources, and optional hands-on challenges.

  <Card title="Browse Workshops" icon="video" href="/workshops">

    Join live workshop sessions
  </Card>
</CardGroup>

## Workshop Structure

```typescript
type Workshop = {
  id: string | number;
  name: string;
  slug: string;
  status: "draft" | "published" | "soon" | "archived" | "streaming";
  difficulty: 1 | 2 | 3;
  duration_in_seconds: number;
  short_description: string;
  description: string;
  instructor: Instructor;
  is_standalone: boolean;
  is_premium: boolean;
  lessons: SidebarLesson[];
  lesson_sections: LessonSection[];
  resources: Resource[];
  tags: Tag[];
}
```

### Workshop Types

<Tabs>
  <Tab title="Standalone Workshops">
    Independent courses marked with a live icon. These are comprehensive workshop sessions recorded or streamed live.
    
    ```typescript
    is_standalone: true  // Workshop icon: RiLiveLine
    ```
  </Tab>
  
  <Tab title="Tutorial Workshops">
    Solution-focused tutorials that teach specific concepts or accompany challenges.
    
    ```typescript
    is_standalone: false  // Tutorial icon: AiOutlineSolution
    ```
  </Tab>
</Tabs>

## Lesson Organization

Workshops are organized into sections with individual lessons:

```typescript
lesson_sections: {
  name: string;
  lesson_ids: number[];
}[];

lessons: SidebarLesson[];
```

<Steps>
  <Step title="Sections">
    Lessons are grouped into logical sections (e.g., "Introduction", "Core Concepts", "Advanced Topics")
  </Step>
  
  <Step title="Individual Lessons">
    Each lesson is a video segment with a specific topic
  </Step>
  
  <Step title="Progress Tracking">
    The platform tracks which lessons you've watched
    
    ```typescript
    first_unwatched_lesson: SidebarLesson | null;
    ```
  </Step>
</Steps>

### Workshop Information

Each workshop displays:

- **Difficulty level**: Visual indicator (1-3)
- **Lessons count**: Total number of video lessons
- **Duration**: Human-readable format (e.g., "2h 30min")
- **Instructor**: Name and avatar with bio on hover
- **Access level**: Free or PRO content indicator

```typescript
<CardItemDifficulty difficulty={workshop.difficulty} />
<CardItemLessonsCount lessonsCount={workshop.lessons.length} />
<CardItemDuration durationString={humanTimeFormat(duration_in_seconds)} />
```

## Enrollment and Progress

### Joining a Workshop

```typescript
await userJoinedWorkshop(slug, request);
// Registers user participation in the workshop
```

### Tracking Progress

```typescript
type WorkshopUser = {
  id: number;
  workshop_id: number;
  user_id: number;
  status: "in-progress" | "completed";
  completed_at: string;
  created_at: string;
  updated_at: string;
  certificate: Certificate;
}
```

<Note>
Your progress is automatically saved as you watch lessons. Return anytime to continue where you left off.
</Note>

## Workshop Cards

Workshops appear as cards in listings:

```typescript
type WorkshopCard = {
  id: string;
  name: string;
  slug: string;
  video_url?: string;
  duration_in_seconds: number;
  status: string;
  is_standalone: boolean;
  is_premium: boolean;
  lessons_count: number;
  instructor: InstructorSummary;
  published_at?: string;
  streaming_url?: string | null;
  pivot?: {
    status: string;
    completed_at: string;
    started_at: string;
    percentage_completed: number;
  };
}
```

<Info>
The `pivot` object tracks your personal progress through the workshop.
</Info>

## Workshop Statuses

<Tabs>
  <Tab title="Published">
    Available for viewing with all lessons accessible
  </Tab>
  
  <Tab title="Soon">
    Scheduled for future release. Shows countdown or scheduled date.
  </Tab>
  
  <Tab title="Streaming">
    Currently live-streaming with real-time viewing
  </Tab>
  
  <Tab title="Draft">
    Not visible to regular users (admin only)
  </Tab>
  
  <Tab title="Archived">
    No longer actively promoted but may still be accessible
  </Tab>
</Tabs>
